Keyboard Shortcuts

  • Up/Down arrow: blend from selected color to the next set color above or below
  • Left: generates shadow gravitating towards the shadow color set in options
  • Right: generates highlight gravitating towards the highlight color set in options
  • t: brings up a menu for quick actions to take on the selected block
  • l: lightens the current color
  • b: brightens the current color
  • d: darkens the current color
  • s: saturates the current color
  • e: desaturates the current color
  • h: shifts the hue by 10 for the current color
  • c: copy the selected color
  • v: paste the copied color
  • delete: deletes the selected color
  • q: shows the menu with options to quit, pin and unpin Palette Mate

Color blending with the up/down arrow requires that you select a color and that there is another color somewhere above or below the selected color. It will blend from one to the other filling in all the blocks in between.

If you press the "Show Options" button you can select the background color and the shadow/highlight color that is used when pressing right/left to generate shadows and highlights.
